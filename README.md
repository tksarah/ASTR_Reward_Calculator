# アプリケーション概要
このリポジトリは、個人開発したアプリケーションを管理するためのものです。  
各ディレクトリには、アプリケーション本体のコードや関連ライブラリを配置しています。

## 含まれているアプリ

### 1. TodoApp
シンプルなタスク管理アプリです。  
ToDo の追加・管理をブラウザ上で手軽に行うことを目的としています。

### 2. ASTR_Reward_Calculator
Astar Network の dApp Staking リワードを記録・集計するための Web アプリです。  
リワードの円換算、月別グラフ表示、JSON / CSV の入出力、ダークモード切り替えなどに対応しています。

# Overview
This repository contains applications developed for personal use.  
Each directory includes the application code and related libraries.

# Applications

## 1. TodoApp
A simple task management application.  
It is intended for lightweight ToDo creation and management in the browser.

## 2. ASTR_Reward_Calculator
A web application for recording and aggregating Astar Network dApp Staking rewards.  
It supports JPY conversion, monthly chart visualization, JSON / CSV import-export, and dark mode switching.
